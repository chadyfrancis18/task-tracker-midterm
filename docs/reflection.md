# Reflection

During the development of this project, AI-assisted coding tools significantly accelerated the process of scaffolding the FastAPI endpoints, writing Pydantic validation models, and setting up the pytest framework. 

One specific moment where AI helped tremendously was in structuring the date comparison logic for the overdue filter without running into timezone or format mismatch errors. Conversely, one moment where it slowed things down slightly was when handling edge cases in frontend DOM manipulation, which required manual debugging and CSS adjustments. 

Overall, leveraging AI as a collaborative partner improved code quality, structure speed, and adherence to clean API design principles.
